# Lezione 1 - Introduzione al Machine Learning

Questo repository contiene due notebook di lezione per l'introduzione al machine learning.

- `Lezione_1_0_ML_ISS2025_introduzione_al_machine_learning.ipynb`
  - Introduce concetti di base di Python e NumPy.
  - Mostra come creare array, matrici, operazioni vettoriali e manipolare tipi di dati.
  - Genera un dataset semplice da usare in un esercizio di classificazione.

- `Lezione_1_1_ML_ISS2025_introduzione_al_machine_learning.ipynb`
  - Mostra un pipeline di apprendimento supervisionato.
  - Crea dataset lineari e non lineari (ad esempio dataset a "concentric circles").
  - Confronta due modelli: regressione logistica e k-NN.
  - Spiega come dividere il dataset in train/validation/test e come valutare le prestazioni.
  - Introduce concetti di feature engineering e validazione incrociata.
